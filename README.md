# pnb_ewallet_api

> API app powered by Ruby on Rails

-------------------------------------------------
### Dependencies
|Library :|Version|
|--------|:------|
|ruby|2.6.1|
|rails|5.2.3|
|postgreSQL|11.2|

-------------------------------------------------
## Run

``` bash
# install dependencies
$ bundle

# create DB
$ rails db:setup

# run server
$ rails s
```

http://pnb-ewallet-api.lvh.me serves the API's
